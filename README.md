# Addressable
- 사용되는 리소스를 Addressable에서 목적에 따라 Group화를 하여 빌드 후 
AWS3에 Bundle을 업로드하여 사용했습니다.
- 실행 시 AWS3 경로에서 다운로드 할 크기를 받아와 다운로드 할 것이 있으면 다운로드를 진행 합니다.
- 그 후 목적에 따라 Handle와 List를 만들어 저장하여 사용합니다.
